# FlorenTak - Portfolio Personnel

Bienvenue sur **FlorenTak**, le portfolio personnel de Florent AYIDEDJI. Ce site présente mes projets, mes compétences et mes informations de contact. Il est construit avec **HTML**, **CSS**, **JavaScript** et **Tailwind CSS** pour un design moderne et réactif.

## Fonctionnalités
- **Section Accueil** : Présentation personnelle et liens vers mes réseaux.
- **Section À propos** : Description de mes compétences et de mon parcours.
- **Section Compétences** : Liste de mes compétences techniques.
- **Section Projets** : Galerie de mes projets récents avec descriptions.
- **Section Contact** : Formulaire de contact pour me contacter directement.

## Technologies utilisées
- **HTML** : Structure de la page.
- **Tailwind CSS** : Styles et design responsive.
- **JavaScript** : Interactivité et animations.
- **Font Awesome** : Icônes modernes.

## Configuration du projet

### Prérequis
- **Node.js** : Assurez-vous d'avoir Node.js installé sur votre machine. Vous pouvez le télécharger [ici](https://nodejs.org/).

### Installation
1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/Florentak21/FlorenTak.git
2. Accédez au dossier du projet :
   ```bash
   cd FlorenTak
3. Installez les dépendances :
   ```bash
   npm install
4. Générez le fichier CSS avec Tailwind CSS :
   ```bash
    npm run build:css
5. Ouvrez le projet avec Live Server :
    Si vous utilisez VS Code, installez l'extension Live Server.

    Cliquez droit sur index.html et sélectionnez Open with Live Server.

    Le site sera accessible à l'adresse http://127.0.0.1:5500.
